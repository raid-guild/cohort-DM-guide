# <img src="/iconography/magic/feather.svg" alt="DM prep checklist icon" className="rg-inline-icon" />7.10 DM Prep Checklist (Quick Reference)

**Before Cohort Starts:**

* Select theme
* Prepare Info Session deck
* Prepare Theme Deep Dive
* Schedule Week 1 sessions
* Set up Discord channels
* Recruit presenters & mentors
* Prep resource links

**During Week 1:**

* Run daily sessions
* Capture ideas
* Facilitate team formation
* Ensure every participant is on a team

**During Async Weeks:**

* Send weekly check-ins
* Monitor progress
* Run office hours as needed

**Before Demo Day:**

* Confirm demos
* Prep judging materials
* Confirm run-of-show
